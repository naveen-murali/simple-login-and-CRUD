# simple-login-and-CRUD

Simple login page and integratting mongoDB.

## Technologies
* node js [v16.13.1]
* express
* monogdb
* connect-mongo
* express-session
* bcrypt
* express-handlebars
* express-validator

### Version
2.0.2

## Usage


### Installation

Install the dependencies

```sh
$ npm install
```
Run in production

```sh
$ npm start
```
Run in development

```sh
$ npm run dev
```
